# Documentation

## 1. Project 1 — Data Analysis with Pandas (PBM)

Gambaran Umum Proyek
Project 1 berfokus pada pengolahan dan analisis data survei menggunakan Python dan Pandas. Data yang dianalisis berkaitan dengan produk bioteknologi yogurt, sehingga proyek ini menggabungkan sains, teknologi, dan data analysis.

Keterkaitan dengan PBM
Proyek ini relevan dengan PBM karena:
- Mengembangkan literasi data
- Melatih analisis berbasis fakta
- Menggunakan teknologi untuk mendukung pembelajaran IPA
- Mengolah data secara mandiri dan sistematis
  
Tujuan Proyek
- Mengolah data survei menggunakan Pandas
- Menghasilkan insight dari data kuantitatif
- Mendukung pengambilan kesimpulan ilmiah
- Mengaitkan data dengan pengembangan produk
  
Proses Analisis Data
- Pengumpulan data survei
- Input data ke dalam DataFrame Pandas
- Pembersihan data (cleaning)
- Analisis frekuensi dan kecenderungan
- Interpretasi hasil
  
Output Proyek
- DataFrame terstruktur
- Ringkasan statistik
- Insight untuk pengembangan produk yogurt
- Artikel sumatif berbasis data
  
Hasil Pembelajaran

Link:
https://www.canva.com/design/DAGyAhtOtH4/XP_Rku-8ZxWsfYFNbplqyQ/edit?utm_content=DAGyAhtOtH4&utm_campaign=designshare&utm_medium=link2&utm_source=sharebutton

- Memahami dasar Pandas
- Menghubungkan data dengan keputusan
- Melatih berpikir kritis dan logis
  
Kesimpulan Proyek
Project ini menunjukkan bahwa analisis data menggunakan Pandas dapat mendukung pembelajaran sains dan membantu menghasilkan keputusan berbasis data dalam pengembangan produk bioteknologi.



## 2. Project 2 — Visualizer App (PBM Technology Project)

Gambaran Umum Proyek
Project 2 merupakan aplikasi visualizer yang dikembangkan sebagai bagian dari Proyek Belajar Mandiri (PBM) dengan fokus pada pemanfaatan teknologi untuk membantu pemahaman konsep melalui visualisasi data atau proses. Aplikasi ini dirancang untuk mengubah informasi abstrak (angka, proses, atau input pengguna) menjadi representasi visual yang lebih mudah dipahami.

Keterkaitan dengan PBM
- Proyek ini selaras dengan tujuan PBM karena:
- Mengembangkan kemampuan belajar mandiri di bidang teknologi
- Melatih pemecahan masalah melalui pemrograman
- Menggabungkan logika, analisis, dan visualisasi
- Menghasilkan produk digital fungsional
  
Tujuan Proyek
- Membuat aplikasi visual yang interaktif
- Membantu pengguna memahami data atau proses melalui tampilan grafis
- Melatih kemampuan pemrograman dan logika algoritmik
- Menerapkan konsep computational thinking
  
Fitur Utama (Konseptual)
- Input data atau parameter dari pengguna
- Pemrosesan data secara real-time
- Visualisasi hasil dalam bentuk grafik, animasi, atau perubahan tampilan
- Antarmuka sederhana dan mudah digunakan
  
Teknologi yang Digunakan
- Python
- Library pendukung visualisasi (bergantung implementasi, misalnya grafik atau canvas)
- Virtual Environment (venv) untuk manajemen dependensi
  
Struktur Proyek (Umum)
- File utama aplikasi (misalnya main.py / app.py)
- Modul pendukung untuk logika pemrosesan
- Folder environment (.venv) untuk dependensi
  
Cara Menjalankan Aplikasi
- Buat virtual environment Python
- Aktifkan environment
- Install library yang dibutuhkan
- Jalankan file utama aplikasi
  
Proses Pengembangan
- Perencanaan ide visualisasi
- Penentuan input dan output
- Implementasi logika program
- Pembuatan tampilan visual
- Pengujian dan perbaikan
  
Hasil dan Pembelajaran
- Link: https://mega.nz/file/J3l30ZZT#xSU2KHofXuobpgXOMo0cfZUlrpuxGL0bnGv71WKMrW8

Melalui proyek ini, penulis memperoleh pemahaman lebih dalam tentang:
- Cara kerja visualisasi data
- Hubungan antara data, logika, dan tampilan visual
- Pentingnya desain antarmuka yang jelas
- Proses pengembangan aplikasi dari ide hingga produk
  
Kesimpulan Proyek
Visualizer App ini menunjukkan bahwa teknologi dapat digunakan sebagai alat bantu belajar yang efektif. Dengan menggabungkan pemrograman dan visualisasi, konsep yang kompleks dapat disederhanakan dan lebih mudah dipahami. Proyek ini memenuhi tujuan PBM dalam mengembangkan kemandirian belajar, kreativitas, dan keterampilan teknologi.



## 3. Project 3 — Soft Skills Coach Website (PBM Website Project)

Gambaran Umum Proyek
Project 3 adalah website Soft Skills Coach yang dirancang untuk membantu pengguna melatih keterampilan non-teknis seperti komunikasi, empati, dan pengembangan diri. Website ini merupakan produk PBM berbasis teknologi dan pengembangan karakter.

Keterkaitan dengan PBM
Proyek ini mendukung PBM karena:
- Menggabungkan teknologi dengan pengembangan karakter
- Menjawab kebutuhan pembelajaran soft skills
- Melatih desain antarmuka dan user experience
- Menghasilkan produk yang bermanfaat secara sosial
  
Tujuan Proyek
- Membuat media pembelajaran soft skills berbasis web
- Membantu pengguna memahami dan melatih soft skills
- Mengembangkan kemampuan front-end development
  
Fitur Utama
- Konten edukatif soft skills
- Tampilan interaktif dan ramah pengguna
- Navigasi sederhana
  
Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- GitHub Pages
  
Link Live Project
https://rethinkksprojects.github.io/soft-skills-coach/

Hasil Pembelajaran
- Memahami hubungan teknologi dan perilaku manusia
- Mengembangkan empati dan komunikasi
- Mendesain aplikasi edukatif
  
Kesimpulan Proyek
Soft Skills Coach Website menunjukkan bahwa teknologi dapat menjadi sarana efektif untuk pengembangan karakter dan pembelajaran non-akademik dalam konteks PBM.
